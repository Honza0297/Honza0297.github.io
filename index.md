A hands-on workshop covering ESP-IDF installation, project creation, component-based development,
Wi-Fi connectivity, NVS data storage, Wi-Fi provisioning, and low-power (ULP) core programming.

Praktický workshop pokrývající instalaci ESP-IDF, tvorbu projektů, vývoj s komponentami,
Wi-Fi konektivitu, NVS ukládání dat, Wi-Fi provisioning a programování low-power (ULP) jádra.

## About / O workshopech

These workshops are designed for developers who want to learn ESP32 development using the official
[ESP-IDF framework](https://github.com/espressif/esp-idf).

**Estimated time:** 180 minutes each  
**Hardware:** ESP32-C5-DevKitC or ESP32-C6-DevKitC  
**Software:** Visual Studio Code with ESP-IDF extension
