## Workshop Overview / Přehled workshopu

This workshop is designed for developers who want to learn ESP32-C6 development using the official ESP-IDF framework. Whether you're a beginner or experienced developer, you'll gain practical skills in IoT development.

**Estimated time:** 180 minutes  
**Hardware required:** ESP32-C6-DevKitC or ESP32-C6-DevKitM  
**Software required:** Visual Studio Code with ESP-IDF extension

### What You'll Learn / Co se naučíte

- 🛠️ **Development Environment Setup** - Install and configure ESP-IDF with VSCode
- 🧩 **Component-Based Development** - Use ESP-IDF components and BSP for rapid development
- 📡 **Wi-Fi Connectivity** - Connect ESP32-C6 to wireless networks and internet
- 💾 **Data Persistence** - Store configuration data using NVS (Non-Volatile Storage)
- 📱 **Mobile Integration** - Configure devices via smartphone using BLE provisioning
- 🔒 **Secure Communication** - Implement HTTPS and TLS for secure data transmission
- ⚡ **Power Optimization** - Utilize the LP core for ultra-low-power applications

### Prerequisites / Předpoklady

- Basic programming knowledge in C
- Computer with Windows, Linux, or macOS
- USB cable compatible with your development kit
- Mobile phone with ESP BLE Provisioning app (for Assignment 5)
